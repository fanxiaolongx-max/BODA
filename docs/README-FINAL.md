## 🎯 核心功能

### 用户端

1. **浏览菜单** - 无需登录，左侧分类，右侧商品
2. **选择规格** - 杯型、甜度、加料
3. **加入购物车** - 底部显示总价
4. **去结算** - 自动提示登录（如未登录）
5. **查看订单** - 显示完整配置
6. **个人中心** - 用户信息和快捷入口

### 管理端

1. **数据概览** - 订单统计
2. **菜品管理** - CRUD，支持图片上传
3. **分类管理** - CRUD
4. **订单管理** - 查看所有订单及配置
5. **系统设置** - 点单开关、货币配置
